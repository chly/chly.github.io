理想的个人MOOC环境

By chly

4月 12 2014 更新日期:5月 27 2014

文章目录

几经寻觅，终于搭建了理想中的个人MOOC环境
能被公网访问，可以发文章，文章能分类，能标签管理，可以放视频，可以放图片，与SNS无缝对接，能接受评论，能查看访客信息。啊，这实在是太好了！并且，用起来很方便！！还是免费的！！！
这种尤物到底是什么呢？
就是本站，由windows+git+Github+Nodejs+hexo+多说+google Analysis+Markdown+sublime打造
Windows 呢，嘿嘿，我不说你也懂。
git 呢，就是现在最流行的版本控制工具。它的资料实在太多，用它的人也实在太多，不信问GOOGLE，这里略过不讲哈
Gighub 呢，程序员们最喜欢用的社交站点，关键时可以挂站点，此站就挂在github上哦。很多大牛栖息于此哦，详见：如何高效利用GitHub。
Nodejs 呢，不得不说，它是当今时代快速开发的利器。还不知道？问百度去。
hexo 呢，据说最初由台湾的一个大学生创造的，现在很多经常写博客的都从原来的阵地转移到这里来了。我追查了下常见的博客使用的环境的进化路线，大致是这样的：

  • 使用公共平台，如新浪、网易、科学网等
  • 建个完全由自己掌控的小站
  • wordpress 现在很多名博还在用这个平台
  • jekyll/octopress
  • hexo

hexo位于进化的最顶端，是刚刚流行不久的东东，用了它，就意味着你是很潮的。
多说 呢，是个评论模块，利用它，hexo就可以接受评论了。至于为什么用它，就不细讲了，统计之都的评论也是用的它。
google analysis 呢， 在hexo上配置了它，你可以知道访问你的站点的都来自己哪里等站点信息
markdown 呢，让你的思路像流水一样转化为文字，写作的时候专注于写作
sublime 呢，经多人验证，一款很好用的文本编辑器。支持markdown语法高亮。

经亲测，视频、图片、链接无障碍。还在愁学校批不批名额给你建mooc?还在愁没钱怎么教改？还在YY最快把你的材料传达给学生的有效方法？
行动起来，hexo吧，这是个想怎么MOOOC就怎么MOOC的环境，没有什么5到25分钟的限制，没有什么名额资源限制。你的电脑和互联网，足矣！
另外，本环境据内容的不同，也可以拓展为专用站点，如
公共微博微信平台小编可以用，微博微信发送两不误
学科馆员可以用，嗯，据我的个人体验，比libguide好用
任课老师可以用，当主页，绝对高大上。你用了，就意味着，你是github的会员，就意味着你还可以建N个库，存各种文档、源码、与别人分享等。。。
辅导员、管理工作者也可以用啦，由它带来的好处，谁用谁最清楚
唉啊，不局限于学校里的工作人员啦，文艺青年、技术青年、科学青年都可以用啊，各有各的玩法，至于怎么玩，网上很多介绍啦
最后，祝大家都能用出自己的范儿！
