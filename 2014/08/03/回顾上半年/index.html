半年回顾

记得去年学校里暑期信息化会议上，有很多代表都做了在线教学相关的报告，当时的讨论也很激烈。当时的热点问题是：浙大要不要像清华北大上交复旦一样，加盟或者加入EDX、Coursera这样的MOOC平台？浙大自已的在线教学接下去要做什么。

去年11月，老大让我摸索下EDX。今年一开学，又成立了在线教学专案组。我有幸成为组内成员。这里简要回顾下这6个月的个人点滴。

  • 对MOOC的认知

MOOC，尤其是在EDX、Coursera上的课程，是一种新型教学形式。灵活互动，没有时间和空间的局限。既然是一种上课形式。课程是否成功，仍然取决于参与者精力和时间的投入程度。课程的目的仍然是让人学习一些东西。就是给一群人注入一些可以让他们改变的东西。在精力和时间给定的情况下，如何做能收好更好的效果，MOOC提供了一种思路。MOOC平台能做的事情是：（1）给老师面向全世界招收学生的机会；（2）给全世界学生、老师互动的机会；（3）据师生历来上课情况，为现在和以后的师生总结出一些怎么样更好地教或者学的经验。

  • 对高校教学的思考

高校是目前高等教育的主体机构。教学是高校的核心使命之一。培养出更优秀的学生是高校共同的目标。培养学生的主要手段时教学。随着社会的进步，工具的革新，教学的方法和手段也在变化。高校的在线教学，是实时实地教学的一种辅助和补充。互联网出现后，在线教学在高校的地位不断提升。在线教学的技术和平台也在不断改进，尤其是到2012年，出现了MOOC这种形式。

  • 技术漫游

那么，在线教学都需要哪些技术呢？

首先，网要通。无论是老师还是学生，都要能访问网络，最好能有可随时随地访问的畅通网络。这需要高校设法建设并高质量维护校园范围内能满足不同学科需求的网络。什么样的网络是个畅通的网络？我这里指一个理想的前满足日常教学使用接入网指标：接入端稳定在网速1Mbps-8Mbps，可用性在99.9%。现在世界各国都在研究如何建一个更好的未来网络。就在今天，来自中国日报的消息称，目前世界上最快的光纤由DTU（丹麦科技大学）研制，网速可达43Gbps，这个速度相当于一毫秒可下载5GB的电影。当然了，校园网建设规划不必追求最近技术，主要考虑点为成本、易用、易管。现行供应商的方案都比较成熟。现在是个数字为王的时代，创新点在于如何在可管上做文章，收集对研究教学有意义的师生上网数据。

第二，资源平台好。无论是老师还是学生，都能随时随地访问到想要的资源，最好能很快发现对教学最有帮助的资源。这需要高校设法建设并高质量维护师生能获取、交流各类资源的平台系列。目前的高校大多有以图书馆为核心的电子资源、以本科生院、研究生院为核心的课程资源、以团委学园为中心的素拓资源、以就业中心校友会为中心的职业人脉资源等。然而这些资源相互独立。目前还不能作为一个整体为师生服务。要把这些资源整合在一起可不是个容易的事儿。以EDX、Coursera为代表的MOOC平台能够以具体的一门课程为主体，将该课程相关的一切联系起来。这给高校教学资源的整合提供了一个新的思路。要整合这些资源，仅凭建立标准和规范是不够的。现在建平台的技术种类太多了，并且还都在进步，标准很难建，建了也不一定有用，即使勉强建起来并且颁布政策要执行了，师生也可能会因讨厌各种不必要的规定而不使用这些平台。这让我想起来“因势利导”这四个字。我们需要建一个这样的平台来整合各类资源：（1）保留现有平台；（2）通过技术手段连接现有平台；（3）接需同步现有资源；（4）对现有资源进行再组合，使用放得易取易用。这有点像BitTorrent中的Tracker，这也是我为什么要把该站点使用为EDU-Tracker的原因。据EDUCAUSE2013的本科生使用信息化的报告显示，课程平台推行不彻底的原因是老师不愿花大量时间，学生因为，没有老师，资源有限也就不上了。我们建这个平台要尽量避免这个问题。这期间涉及的技术包括Web开发、运维、视频资源制作、视频在线服务、数据共享、数据分析、网站分析、网络资源爬取及分析等。
第三，平台要好。

能用，不同分辨率的显示屏，不同的浏览器都能用。这要求门户是Web形式，且页面是屏幕自适应的。因为APP、应用程序的形式设备依赖性太强了。平台建设方可以控制的图片资源是矢量的。视频资源可以网络环境自适应。

易用，师生不需要培训，也不需要过多额外操作，就能做相应的事情

好用，不容易被攻击，被恶意使用，这就要充分考虑安全性及版权访问许可。

能够被大规模使用，这需要考虑并发量，需要考虑负载均衡等。

  • 教育&信息

教育的目的是培养人。培养人又可分为三个方面（1）传递知识;(2)传授思想；（3）改进修养。 这三个方面的增长都是以注入信息为手段的。所以，我认为，实施教育的本质，就是引导人给自己注入信息的艺术。在信息技术发达的当代，信息技术在教育领域大有可为。

  • 具体事件

这半年，我个人对信息技术如何支撑高校教育做了大量启蒙性探索。共记在线教学相关笔记166篇，未来网络8篇，图书馆10篇，数据科学69篇，未明确归类103篇。阅读教育学教材一本，选看多贝网、可可英语网、EDX、Coursera上课程多科。与小组成员一起搭建了edx.zju.edu.cn站点，对Django Web 开发略有探索，Mysql，MongoDB，git有了少量使用经验。对处理数据的R、Python相关包和模块进行了探索性使用。还用基于nodejs开发的hexo搭建了该站。
